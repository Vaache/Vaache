## About

I build mobile applications with Flutter — clean, scalable, and production-ready. My focus sits at the intersection of solid architecture and real performance: state management that doesn't collapse under complexity, APIs that integrate cleanly, and code that doesn't turn into a liability six months later.

On the backend side, I work with Node.js and TypeScript, with PostgreSQL as the primary data layer. I care about system design — not as a buzzword, but as an engineering practice that shapes every decision.

- Prefer **BLoC** where predictability matters
- Think in **Clean Architecture** — separation of concerns isn't optional
- Currently deepening **performance optimization** and **distributed system patterns**
