Flutter developer with a **system programming foundation** (C/C++, 42 Yerevan). I build cross-platform mobile and desktop applications where architecture, performance, and maintainability matter equally — not just for the first release, but six months later.

- **Native depth:** BLE via platform channels, push notifications, iOS/Android notification channels
- **Real-time systems:** WebSocket streaming, live video (Tencent LiveKit), synchronized multi-window UI
